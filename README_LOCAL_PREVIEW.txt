GPUN homepage prototype local preview

How to view:
1. Extract this zip.
2. Open index.html in a browser.
3. For the most reliable Media page behavior, you can also run a tiny local server from this folder:
   python -m http.server 4173
   then open http://127.0.0.1:4173/

This package includes only files referenced by the site, using optimized gallery images and optimized page videos.
